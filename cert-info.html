Ringkasan Materi

Ingin mendapatkan kemampuan pengujian perangkat lunak untuk memulai karir atau apakah Anda seorang pengembang perangkat lunak yang ingin meningkatkan kempuan pengujian unit anda ? Kursus ini, bagian dari program pengujian Perangakat Lunak dan Verifikasi MikroMaster, akan memberikan keterampilan penting yang anda butuhkan untuk sukses.

Perangkat lunak harus diuji untuk bug dan memastikan produk memenuhi persyaratan dan menghasilkan hasil yang diinginkan. Pengujian perangakat lunak sangat penting untuk menyediakan produk berkualitas.

Pelajari teknik Software Testers dan Quality Assurance Engineers gunakan tiap hari, yang dapat diterapkan ke bahasa pemrograman dan perangkat lunak pengujian.

Tidak Diperlukan pengetahuan pemrograman sebelumnya. Kursus in akan menggunakan java dan JUnit, namun, untuk contoh dan tugas.

Kursis ini merupakan bagian dari pengujian Perangkat Lunak dan program Verifikasi MicroMaster dari UMUC. Setelah menyelesaikan program dan menerima sertifikat MicroMaster yang Terverifikasi, pelajar kemudian dapat beralih ke program master UMUC penuh di Teknology Informasi dengan spesialisasi dalam Rekayasa Perangkat Lunak tanpa proses aplikasi atau pengujian apa pun. Luhat Halaman Program MicroMaster untuk lebih lanjut.


Jadwal

  • Hari 1
    Kembangkan, Dokumentasikan, dan laksanakan uji Perangkat Lunak


  • Hari 2
    Buat tes perangkat lunak untuk memenuhi tujuan


  • Hari 3
    Identifikasi jenis cacat perangkat lunak
